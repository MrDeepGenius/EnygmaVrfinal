# 🚀 QUICK START - Generar APK en 5 Minutos

## ⚠️ Requisitos (descarga estos primero):

1. **Android Studio** → https://developer.android.com/studio
2. **JDK 11+** → https://www.oracle.com/java/technologies/javase-jdk11-downloads.html
3. **Node.js** (ya tienes) ✅

---

## ⚡ Instrucciones Rápidas

### Paso 1: Instala Capacitor
```bash
npm install -g @capacitor/cli
```

### Paso 2: Abre PowerShell en la carpeta del proyecto
```bash
cd "c:\Users\gabii\Downloads\ULTIMO APP CO\levantar (1)\artifacts\enygma"
```

### Paso 3: Agrega plataforma Android
```bash
npx cap add android
```

### Paso 4: Sincroniza
```bash
npx cap sync android
```

### Paso 5: Abre Android Studio
```bash
npx cap open android
```

### Paso 6: En Android Studio
1. Espera a que se abra
2. Click en: **Build** (menú superior)
3. Click en: **Generate Signed Bundle/APK**
4. Elige: **APK**
5. Elige: **Release**
6. Crea un archivo keystore (o usa uno existente)
7. ¡Completa y listo! 🎉

---

## 📁 Dónde estará el APK:

```
artifacts\enygma\android\app\release\app-release.apk
```

---

## 🎯 Para enviar a TV:

1. Copia el APK a un USB
2. Conecta USB a la TV
3. Abre: Settings → Security → Unknown Sources (ON)
4. Instala el APK
5. ¡Abre ENYGMA en la TV! 🎬
